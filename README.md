The Student Record Management System (SRMS) is a Python-based command-line application with an SQLite database backend designed to manage student information efficiently. It allows users to perform full CRUD operations including adding, viewing, searching, updating, and deleting student records.

The system stores data permanently in a local database (students.db) and ensures data integrity through proper validation and exception handling. It also features a clean, formatted table display for better readability of records in the terminal.

This project demonstrates key programming concepts such as Python functions, SQL queries, database connectivity, data validation, and modular code structure, making it a strong foundation for learning real-world database-driven applications.
